# Yamini Coen

A Y2K resume website for Yamini Coen, a communications, public relations and digital marketing expert based out of Toronto, Canada.

[YaminiCoen.com](https://yaminicoen.com/)

Original project made by [@danunder](https://github.com/danunder) using a template from [@timbakerx](https://github.com/tbakerx/). Most of the project if built around the design from [98.css](https://jdan.github.io/98.css/).

## Installation

```bash
yarn install
```

## Usage

```bash
yarn dev
```

## Studio

Navigate to `/studio` of your domain to update/edit the sanity content once the server is running.
